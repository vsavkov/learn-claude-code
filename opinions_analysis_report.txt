================================================================================
OPINION TYPE DESCRIPTIONS
================================================================================

1. MAJORITY (95.0%)
   - The main opinion of the court representing the majority view
   - Contains the court's reasoning and decision
   - Field: opinions.text contains the full text
   - Author field often populated with judge name

2. DISSENT (2.5%)
   - Written by judges who disagree with the majority decision
   - Presents alternative reasoning and conclusion
   - Often longer and more detailed than majority opinions
   - Examples show dissents can be 30,000+ characters

3. CONCURRENCE (1.8%)
   - Written by judges who agree with the result but for different reasons
   - Or agree with part of the majority's reasoning
   - Generally shorter than majority opinions

4. CONCURRING-IN-PART-AND-DISSENTING-IN-PART (0.7%)
   - Judge agrees with some aspects but disagrees with others
   - Hybrid opinion type
   - Provides nuanced perspective on complex cases

5. REMITTITUR (0.0%)
   - Extremely rare (1 instance in 50,000 cases)
   - Order to reduce damages awarded by jury
   - Special procedural opinion type
